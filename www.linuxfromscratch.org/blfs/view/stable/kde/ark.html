Ark-4.14.1

Introduction to Ark

This package provides an archiving utility for KDE.

This package is known to build and work properly using an LFS-7.6 platform.

Package Information

  • Download (HTTP): http://download.kde.org/stable/4.14.1/src/ark-4.14.1.tar.xz

  • Download MD5 sum: 55fcc756e600e18cdc8bfc5c562657a8

  • Download size: 232 KB

  • Estimated disk space required: 12.2 MB

  • Estimated build time: 0.3 SBU

Ark Dependencies

Required

kde-baseapps-4.14.1 and libarchive-3.1.2

Optional

QJson-0.8.1

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/ark

Installation of Ark

Install Ark by running the following commands: 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KDE_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -Wno-dev .. &&
make

Now, as the root user: 

make install

Contents

Installed Program: ark
Installed Libraries: libkerfuffle.so and several in $KDE_PREFIX/lib/kde4
Installed Directories: several in $KDE_PREFIX/share

Short Descriptions

ark

is the KDE archiving utility.
