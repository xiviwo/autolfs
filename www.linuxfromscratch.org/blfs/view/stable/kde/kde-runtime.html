Kde-runtime-4.11.1

Introduction to Kde-runtime

Kde-runtime contains runtime applications and libraries essential for KDE.

This package is known to build and work properly using an LFS-7.4 platform.

Package Information

  • Download (HTTP): http://download.kde.org/stable/4.11.1/src/kde-runtime-4.11.1.tar.xz

  • Download (FTP): ftp://ftp.kde.org/pub/kde/stable/4.11.1/src/kde-runtime-4.11.1.tar.xz

  • Download MD5 sum: b195a10b9ffda33aaef5b34092ddc23a

  • Download size: 7.6 MB

  • Estimated disk space required: 111 MB

  • Estimated build time: 7.5 SBU

Additional Downloads

  • Required patch: http://www.linuxfromscratch.org/patches/blfs/svn/kde-runtime-4.11.1-rpc_fix-1.patch

Kde-runtime Dependencies

Required

kdelibs-4.11.1 and libtirpc-0.2.3

Recommended

kactivities-4.11.1, kdepimlibs-4.11.1, alsa-lib-1.0.27.2, libjpeg-turbo-1.3.0 and Exiv2-0.23

Optional

GDB-7.6.1, PulseAudio-4.0, xine-lib-1.2.3, libcanberra-0.30, Samba-4.0.9, NetworkManager-0.9.8.2, OpenSLP, QNtrack, LibSSH and OpenEXR

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/kde-runtime

Installation of Kde-runtime

Install kde-runtime by running the following commands: 

patch -Np1 -i ../kde-runtime-4.11.1-rpc_fix-1.patch &&

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KDE_PREFIX \
      -DSYSCONF_INSTALL_DIR=/etc         \
      -DCMAKE_BUILD_TYPE=Release         \
      .. &&
make

To test the results, issue make test.

Now, as the root user: 

make install &&
ln -s -v ../lib/kde4/libexec/kdesu $KDE_PREFIX/bin/kdesu
