Kactivities-4.12.2

Introduction to Kactivities

This package provides the activitymanager for KDE.

This package is known to build and work properly using an LFS-7.5 platform.

Package Information

  • Download (HTTP): http://download.kde.org/stable/4.12.2/src/kactivities-4.12.2.tar.xz

  • Download (FTP): ftp://ftp.kde.org/pub/kde/stable/4.12.2/src/kactivities-4.12.2.tar.xz

  • Download MD5 sum: b446703c2e6d85427e8c7dd7e589bed5

  • Download size: 85 KB

  • Estimated disk space required: 10.8 MB

  • Estimated build time: 0.9 SBU

Kactivities Dependencies

Required

kdelibs-4.12.2

Recommended

nepomuk-core-4.12.2

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/kactivities

Installation of Kactivities

Install Kactivities by running the following commands: 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KDE_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -Wno-dev .. &&
make

This package does not ship a default test suite.

Now, as the root user: 

make install

Contents

Installed Programs: kactivitymanagerd
Installed Libraries: libkactivities.so, libkactivities-models.so and several in $KDE_PREFIX/lib/kde4
Installed Directories: $KDE_PREFIX/include/kactivities, $KDE_PREFIX/include/kactivities-models, $KDE_PREFIX/include/KDE/KActivities, $KDE_PREFIX/lib/cmake/KActivities, $KDE_PREFIX/lib/cmake/KActivities-models, and several in $KDE_PREFIX/share
