IBus-1.5.1

Introduction to IBus

IBus is an Intelligent Input Bus. It is a new input framework for Linux OS. It provides full featured and user friendly input method user interface.

This package is known to build and work properly using an LFS-7.2 platform.

Package Information

  • Download (HTTP): http://ibus.googlecode.com/files/ibus-1.5.1.tar.gz

  • Download MD5 sum: f0103201249c657712f5f4d9a36923ed

  • Download size: 1.9 MB

  • Estimated disk space required: 50 MB

  • Estimated build time: 0.5 SBU

IBus Dependencies

Required

DConf-0.14.1 and ISO Codes-3.40

Recommended

gobject-introspection-1.34.2 and Vala-0.18.1

Optional

D-Bus Python Bindings-1.1.1, GTK+-2.24.17, GTK-Doc-1.18 and pyxdg

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/ibus

Installation of IBus

Install IBus by running the following commands: 

./configure --prefix=/usr \
            --sysconfdir=/etc \
            --libexecdir=/usr/lib/ibus \
            --disable-gtk2 &&
make

To test the results, issue: make check.

Now, as the root user: 

make install

Command Explanations

--disable-gtk2: This switch disables building of the GTK+ 2 immodule. Remove it if you have installed GTK+ 2 and wish to build the GTK+ 2 immodule.

--enable-gtk-doc: Use this parameter if GTK-Doc is installed and you wish to rebuild and install the API documentation.

Contents

Installed Programs: ibus, ibus-daemon and ibus-setup
Installed Library: libibus-1.0.so and im-ibus.so (GTK+ Immodule)
Installed Directories: /etc/dconf/db/ibus.d, /usr/include/ibus-1.0, /usr/lib/ibus, /usr/share/gtk-doc/html/ibus and /usr/share/ibus

Short Descriptions

ibus-daemon

is the Intelligent Input Bus Daemon.

ibus-setup

is the GTK+ program used to configure the ibus-daemon.

libibus-1.0.so

contains IBus API functions.
