appdata-tools-0.1.7

Introduction to appdata-tools

The appdata-tools is used to validate AppData files to conform to standard specification.

This package is known to build and work properly using an LFS-7.5 platform.

Package Information

  • Download (HTTP): http://people.freedesktop.org/~hughsient/releases/appdata-tools-0.1.7.tar.xz

  • Download MD5 sum: adc82ce1d396c45203a842a508114ead

  • Download size: 260 KB

  • Estimated disk space required: 2.7 MB (additional 0.3 MB for the tests)

  • Estimated build time: less than 0.1 SBU

appdata-tools Dependencies

Required

gdk-pixbuf-2.30.7, Intltool-0.50.2, and libsoup-2.46.0

Recommended

gobject-introspection-1.40.0

Optional

libxml2-2.9.1, libxslt-1.1.28, docbook-xml-4.5, docbook-xsl-1.78.1, trang, and lxml

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/appdata-tools

Installation of appdata-tools

Install appdata-tools by running the following commands: 

./configure --prefix=/usr &&
make

To test the results, issue: make check.

Now, as the root user: 

make install

Installed Libraries: None
Installed Directories: None
