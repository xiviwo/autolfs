Alacarte-3.6.1

Introduction to Alacarte

Alacarte is a menu editor for GNOME Desktop using the freedesktop.org menu specification.

This package is known to build and work properly using an LFS-7.2 platform.

Package Information

  • Download (HTTP): http://ftp.gnome.org/pub/gnome/sources/alacarte/3.6/alacarte-3.6.1.tar.xz

  • Download (FTP): ftp://ftp.gnome.org/pub/gnome/sources/alacarte/3.6/alacarte-3.6.1.tar.xz

  • Download MD5 sum: 4b801fa3125b82d0a29b4ff83d9e9d5b

  • Download size: 176 KB

  • Estimated disk space required: 3.4 MB

  • Estimated build time: less than 0.1 SBU

Alacarte Dependencies

Required

gnome-menus-3.6.2 and PyGObject-3.4.2

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/alacarte

Installation of Alacarte

Install Alacarte by running the following commands: 

./configure --prefix=/usr &&
make

This package does not come with a test suite.

Now, as the root user: 

make install

Contents

Installed Program: alacarte
Installed Libraries: None
Installed Directories: /usr/lib/python2.7/site-packages/Alacarte and /usr/share/alacarte

Short Descriptions
