Gcr-3.12.0

Introduction to Gcr

The Gcr package contains libraries used for displaying certificates and accessing key stores. It also provides the viewer for crypto files on the GNOME Desktop.

This package is known to build and work properly using an LFS-7.5 platform.

Package Information

  • Download (HTTP): http://ftp.gnome.org/pub/gnome/sources/gcr/3.12/gcr-3.12.0.tar.xz

  • Download (FTP): ftp://ftp.gnome.org/pub/gnome/sources/gcr/3.12/gcr-3.12.0.tar.xz

  • Download MD5 sum: e3f729d63b9b53ef99b5c19958bfb6c8

  • Download size: 1.3 MB

  • Estimated disk space required: 62 MB (additional 3 MB for tests)

  • Estimated build time: 0.7 SBU (additional 0.1 SBU for tests)

Gcr Dependencies

Required

GLib-2.40.0, Intltool-0.50.2, libgcrypt-1.6.1, libtasn1-3.4, and p11-kit-0.20.2

Recommended

GnuPG-2.0.22, gobject-introspection-1.40.0, GTK+-3.12.1, libxslt-1.1.28, and Vala-0.24.0

Optional

GTK-Doc-1.20 and Valgrind-3.9.0

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/gcr

Installation of Gcr

Install Gcr by running the following commands: 

./configure --prefix=/usr     \
            --sysconfdir=/etc &&
make

A session bus address is necessary to run the tests. To test the results, issue: make check.

Now, as the root user: 

make install

Command Explanations

--without-gtk: Use this switch if you haven't installed GTK+-3.12.1.

--enable-gtk-doc: Use this parameter if GTK-Doc is installed and you wish to rebuild and install the API documentation.

Contents

Installed Program: gcr-viewer and gcr-prompter
Installed Libraries: libgck-1.so, libgcr-3.so, libgcr-base-3.so and libgcr-ui-3.so
Installed Directories: /usr/include/gck-1, /usr/include/gcr-3, /usr/share/gcr-3, /usr/share/gtk-doc/html/gck, and /usr/share/gtk-doc/html/gcr-3

Short Descriptions

gcr-prompter

provides the prompt dialog needed by libgcr.

gcr-viewer

is used to view certificate and key files.

libgck-1.so

contains GObject bindings for PKCS#11.

libgcr-3.so

contains functions for high level crypto parsing.
