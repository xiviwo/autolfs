Epiphany-3.12.0

Introduction to Epiphany

Epiphany is a simple yet powerful GNOME web browser targeted at non-technical users. Its principles are simplicity and standards compliance.

This package is known to build and work properly using an LFS-7.5 platform.

Package Information

  • Download (HTTP): http://ftp.gnome.org/pub/gnome/sources/epiphany/3.12/epiphany-3.12.0.tar.xz

  • Download (FTP): ftp://ftp.gnome.org/pub/gnome/sources/epiphany/3.12/epiphany-3.12.0.tar.xz

  • Download MD5 sum: 19f959a7fbd3b38bf6d1726e5624d967

  • Download size: 2.9 MB

  • Estimated disk space required: 91 MB

  • Estimated build time: 0.9 SBU

Epiphany Dependencies

Required

Avahi-0.6.31, Gcr-3.12.0, gnome-desktop-3.12.1, libnotify-0.7.6, libwnck-3.4.7, and WebKitGTK+-2.4.0

Recommended

NSS-3.16

Optional

lsb_release-1.4

Runtime Dependencies

gnome-keyring-3.12.0 (for storing passwords) and Seahorse-3.12.0 (for managing stored passwords)

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/epiphany

Installation of Epiphany

Install Epiphany by running the following commands: 

./configure --prefix=/usr &&
make

This package does not come with a testsuite.

Now, as the root user: 

make install

Contents

Installed Program: epiphany, ephy-profile-migrator, and epiphany-search-provider
Installed Libraries: /usr/lib/epiphany/3.12/web-extensions/libephywebextension.so
Installed Directories: /usr/lib/epiphany, /usr/share/epiphany, and /usr/share/help/*/epiphany

Short Descriptions

epiphany

is a GNOME web browser based on the WebKit2 rendering engine.
