Ark-4.13.0

Introduction to Ark

This package provides an archiving utility for KDE.

This package is known to build and work properly using an LFS-7.5 platform.

Package Information

  • Download (HTTP): http://download.kde.org/stable/4.13.0/src/ark-4.13.0.tar.xz

  • Download (FTP): ftp://ftp.kde.org/pub/kde/stable/4.13.0/src/ark-4.13.0.tar.xz

  • Download MD5 sum: bdc9077cca769bdf676c2892651aa17b

  • Download size: 225 KB

  • Estimated disk space required: 9.9 MB

  • Estimated build time: 0.4 SBU

Ark Dependencies

Required

kde-baseapps-4.13.0 and libarchive-3.1.2

Optional

QJson-0.8.1

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/ark

Installation of Ark

Install Ark by running the following commands: 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KDE_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -Wno-dev .. &&
make

Now, as the root user: 

make install

Contents

Installed Program: ark
Installed Libraries: libkerfuffle.so and several in $KDE_PREFIX/lib/kde4
Installed Directories: several in $KDE_PREFIX/share

Short Descriptions

ark

is the KDE archiving utility.
